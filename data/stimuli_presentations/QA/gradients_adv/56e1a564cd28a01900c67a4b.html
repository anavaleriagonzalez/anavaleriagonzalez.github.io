Question:

how is worst - case time complexity written as an expression ?

Context:

if the input size is n , the time taken can be expressed as a function of n . since the time taken on different inputs of the same size can be different , the worst - case time complexity t ( n ) is defined to be the maximum time taken over all inputs of size n . if t ( n ) is a polynomial in n , then the algorithm is said to be a polynomial time algorithm . co bham ' s thesis says that a problem can be solved with a feasible amount of resources if it admits a polynomial time algorithm .
